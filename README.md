## рџ”§ Configuration


Pull from Vercel development env:
```bash

```

### PWA Manifest
**public/manifest.webmanifest** вЂ” Defines:
- App name, description, icons
- Home screen theme color, background color
- Start URL, scope, display mode
- Categories for app stores

See [PWA Setup Guide](./docs/PWA_SETUP.md) for details.

### Service Worker (Serwist)
**src/app/sw.ts** вЂ” Caching strategy:
- **Network-first** for HTML routes (offline fallback в†’ `/offline`)
- **Cache-first** for CSS, JS, fonts, images
- **NetworkOnly** for API calls (5s timeout)
